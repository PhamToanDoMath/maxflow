1. Please run this before run this script:
pip install networkx pandas matplotlib openpyxl

2. Run the script
python maxflow.py

3. Enter the start node and end node, node number is correspond with id of address name. Default value 0 for start node at "Ngã tư An Sương" and end node at "Công viên Thảo Cầm Viên"
For example:
1	AEON MALL Tân Phú
2	Ba Thien Hau Temple
3	Bến xe Miền Tây
